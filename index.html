Easy = fully conversational, nose breathing. Tempo = comfortably hard — you can speak a sentence but don't want to. Long run = aerobic cruise, never gasping. Paces improve naturally as fitness builds — these are starting benchmarks.
Rules
Drive days = full rest, no exceptions.
Fairfield work days (10.5hr shifts) = run only if afternoon/PM shift — morning window only. Skip if morning/unknown shift.
Yoga = hot yoga AM via ClassPass (book early). Run = PM/evening on yoga days.
Leg days = off weeks only. Work weeks = upper body at lunch only.
